# Mini Pi 이족보행 로봇 하드웨어 개요

Mini Pi 이족보행 로봇은 NPU가 탑재된 소형 싱글 보드 컴퓨터(SBC)를 두뇌로 하여, 사람처럼 두 다리를 이용해 걷거나 움직이도록 설계된 플랫폼입니다. 기존의 바퀴형 로봇과 달리 복잡한 균형 및 동역학 제어가 핵심 기술입니다.

Mini Pi는 강력한 연산 능력과 Linux OS 환경을 제공하여, 다음과 같은 첨단 이족보행 알고리즘을 구현하는 데 최적화되어 있습니다.

::: info Support inference by internal NPU
* 동적 균형 유지: IMU 센서 데이터를 이용한 실시간 자세 제어

* 보행 패턴 생성: 걷기, 뛰기, 방향 전환 등 복잡한 움직임 계획

* 고수준 의사 결정: 주변 환경 인지 및 다음 행동 결정
:::

이족보행 로봇은 정밀한 각도 제어를 위해 다수의 서보 모터와 안정적인 제어 시스템을 요구하며, 하드웨어 구성 요소는 이 복잡한 움직임을 지원하는 데 집중되어 있습니다.

핵심 구성 요소는 다음과 같습니다.

::: info Robot System
1. 제어 시스템 : 고수준의 보행 알고리즘 및 통신 허브 역할

2. 구동계 (고출력 서보 액추에이터): 관절의 움직임과 힘을 정밀하게 제어

3. 입력 장치 (IMU, 센서): 로봇의 기울임과 가속도를 실시간으로 감지하여 균형 유지

4. 기계 기구부: 관절의 자유도(DOF)와 안정적인 무게 중심을 고려한 신체 구조
:::
